# Copy Trading

Heliora allows users to subscribe to specialized AI trading agents and automatically mirror their positions on-chain. This democratizes access to advanced algorithmic strategies.

## How It Works

1.  **Browse**: Users can view the performance history of different AI agents (e.g., Momentum Agent, Arbitrage Agent).
2.  **Subscribe**: Users deposit SOL or USDC into a non-custodial "Follower Vault" controlled by the Heliora program.
3.  **Mirror**: When the primary agent executes a trade, the protocol automatically executes a proportional trade for all subscribed followers.

## Position Sizing

The protocol uses a **Ratio-Based Sizing** model. If an agent trades 1% of its total bankroll, the protocol will trade 1% of your subscribed vault balance.

```typescript
// sizing formula
user_trade_amount = (agent_trade_amount / agent_total_balance) * user_vault_balance;
```

## Risk Management

- **Max Drawdown Limit**: Users can set a hard limit (e.g., 20%). If the agent loses more than this, the subscription is automatically canceled and funds are returned.
- **Stop Loss**: Each mirrored position can have an optional individual stop-loss trigger.

## How to Subscribe via SDK

You can programmatically subscribe to an agent using the SDK:

```typescript
await sdk.subscribeToAgent({
  agentId: "momentum-bot-01",
  amount: 10, // SOL
  maxDrawdown: 0.15 // 15%
});
```

## Fee Structure

Heliora operates on a **Performance-Only** fee model:

- **Management Fee**: 0% (Always free to subscribe).
- **Performance Fee**: 10% of net profits generated by the agent. If the agent doesn't make you money, we don't take a fee.
