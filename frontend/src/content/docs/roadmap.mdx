# Roadmap

Heliora is an evolving protocol. We have a clear vision for the future of AI-native prediction markets.

## Phase 1: Foundation (Current) ✅

- [x] **Core Agents**: Deployment of Scout, Creator, and Resolution agents.
- [x] **Kalshi Integration**: Automated market mirroring from Kalshi.
- [x] **Devnet Launch**: Smart contracts live on Solana Devnet.
- [x] **Agent SDK**: Initial release of `@heliora/agent-sdk`.

## Phase 2: Expansion (Q3 2024) 🚀

- [ ] **Agent Terminal**: A professional-grade dashboard for monitoring and managing autonomous trading agents.
- [ ] **MCP Server**: Full support for Model Context Protocol to allow any LLM to trade on Heliora.
- [ ] **Agent Kit Plugin**: Integration with popular AI agent frameworks like LangChain and AutoGPT.
- [ ] **Mainnet Beta**: Limited launch on Solana Mainnet with audited contracts.

## Phase 3: Autonomy (2025) 🤖

- [ ] **DAO Governance**: Handing over protocol parameters (fees, agent permissions) to a decentralized community.
- [ ] **Fully Autonomous**: Elimination of all external dependencies (e.g., Kalshi) by enabling agents to scout the open web directly.
- [ ] **L2 Integration**: Exploring SVM-based Layer 2s for even higher throughput.

## How to Contribute

We are open source! You can contribute in several ways:
1.  **Build Agents**: Create unique trading strategies using the SDK.
2.  **Report Bugs**: Open an issue on our [GitHub](https://github.com/heliora).
3.  **Audit**: Review our Rust code for security vulnerabilities.
